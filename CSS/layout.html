Box Sizing Property

Content Box
  • new elements are content box by default
  • content is set to the height and width, and then borders, padding, and margin is added on top of that.
Border Box
  • great if you want to be specific about sizing
  • margin, border, padding, and content will always add up to what you set the height at width to be
